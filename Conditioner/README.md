❄️ Кондиционер — Температура

Логический сценарий для SprutHub. Управляет кондиционером по температуре в комнате — включает, выключает и плавно регулирует скорость вентилятора. Учитывает присутствие людей, открытые окна, время суток, уличную температуру и ручное управление.

Принцип работы — порядок проверок

1 Зима / Лето Если переключатель «Зима» включён — вся логика остановлена, кондиционер не трогается. Подходит для отключения сценария на холодный сезон без его удаления.

2 Режим охраны (хаб СХ) При режиме «Не дома» кондиционер выключается. «Дома» и «Ночь» — сценарий работает, при ночи применяются ночные пороги температуры и скорости.

3 Открытые окна Если хотя бы одно из четырёх окон открыто — кондиционер немедленно выключается. При закрытии всех окон работа возобновляется при выполнении остальных условий.

4 Уличная температура Если на улице холоднее заданного минимума (по умолчанию 15°C) — кондиционер не включается. Исключает работу в прохладную погоду когда охлаждение не нужно.

5 Присутствие людей Проверяются два уровня: кто-то из указанных людей дома (персональные датчики), и есть ли кто-то в самой комнате (ДП). При уходе из комнаты — таймер задержки перед выключением.

6 Температура и скорость Основная логика: включить при достижении верхнего порога, выключить при нижнем. Скорость пересчитывается каждую минуту по текущей температуре — от «тихо» до «турбо».

7 Дисплей Днём дисплей кондиционера включается, ночью выключается. Управляется через отдельный Switch-сервис.

Настройки — что и зачем

Комната Название для логов: Спальня, Гостиная, Детская и т.д. Помогает различать события разных кондиционеров в общем логе.

Кондиционер (Thermostat) Thermostat-сервис кондиционера в SprutHub. К нему привязан сценарий — он читает температуру и управляет режимом и скоростью.

Датчик температуры Внешний датчик температуры в комнате. Именно по его показаниям принимаются все решения о включении, выключении и скорости.

Целевая температура Значение которое передаётся кондиционеру как цель охлаждения. Отдельно для дня и ночи. Например: день 24°C, ночь 25°C.

Порог включения При достижении этой температуры кондиционер включается. Отдельно для дня и ночи. Например: день 26°C, ночь 25°C.

Порог выключения Гистерезис: при снижении до этой температуры кондиционер выключается. Должен быть ниже порога включения чтобы избежать частых циклов.

Ступени скорости (5 штук) Задаёте верхнюю границу температуры для каждой ступени: тихо → медленно → средне → быстро → турбо. Скорость пересчитывается каждую минуту — снижается сама при охлаждении.

Сигнализация (хаб СХ) SecuritySystem-сервис. Режим «Не дома» — кондиционер выключен. «Ночь» — применяются ночные настройки. «Дома» — стандартная работа.

Переключатель Зима/Лето Switch-сервис. ON = зима, вся логика выключена. OFF = лето, сценарий работает. Удобно чтобы отключить кондиционирование на зиму одним нажатием.

Присутствие людей дома (до 3) OccupancySensor для каждого жильца. Если никого дома — кондиционер не включается. Достаточно чтобы хотя бы один был дома.

ДП в комнате Датчик присутствия в конкретной комнате. Кондиционер включается только если есть движение. При уходе — таймер задержки перед выключением.

Задержка выключения Ползунок вкл/выкл + количество минут. Если включено: при уходе из комнаты кондиционер ждёт N минут и только потом выключается. Если выключено — выключает сразу.

Датчики окон (до 4) ContactSensor для каждого окна. Если хотя бы одно открыто — кондиционер немедленно выключается. При закрытии всех работа возобновляется.

Уличная температура Внешний датчик + минимальный порог. Если на улице холоднее порога — кондиционер не включается. Полезно в межсезонье.

Ночной режим Время начала и конца ночи (ЧЧ:ММ). В ночное время применяются отдельные пороги включения/выключения и ступени скорости — обычно тише и мягче.

Дисплей (Switch) Switch-сервис дисплея кондиционера. Днём включается, ночью выключается автоматически.

Динамическое ускорение Если температура не снижается N минут — скорость автоматически повышается на одну ступень. При снижении температуры — скорость возвращается по расчёту.

Пауза после ручного выключения Если выключить кондиционер вручную — сценарий не включит его автоматически N минут. По истечении паузы возвращается к автоуправлению. 0 = ждать до достижения tOff.

Диагностический лог Boolean-ползунок. При включении каждую минуту пишет в лог: температуру, пороги, состояние AC, текущую скорость и boost. Включать только при отладке.
